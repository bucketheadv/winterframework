## Install

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

```xml
<dependency>
    <groupId>com.github.bucketheadv</groupId>
    <artifactId>winterframework</artifactId>
    <version>v0.3.0</version>
</dependency>
```
